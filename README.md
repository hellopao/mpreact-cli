# mpreact-cli

 cli for mpreact

## install

 ```bash
 npm install mpreact-cli -g
 ```

## usage

 ### init project

 ```bash
 mpreact init example
 ```

 ### build project

 ```bash
 mpreact build -s example/src -d example/dist
 ```

## options

 ### init

 * -l
  
  language for project: ts/js, default to ts

### build

 * -w
    
  	run in watch mode.
